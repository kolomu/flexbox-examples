**Why Flexbox?**
* Flexible Box Model
* Arrange items in container
* No Floats
* Responsive & Mobile friendly
* Order of elements can be changed

**Concepts**
* Ability alter width & height 
* Direction-Agnostic (Horizontal/Vertically)
* For small scaled layouts
